# 🔄 AI CLI Converter — "Woon Translator" for Computer Users!

Have you ever run into this problem? You follow an online tutorial that tells you to type `ls` or `rm -rf` (Linux commands), but your machine is Windows. When you type the commands, your computer angrily replies, "Command not found!" 😫

Solve that instantly with AI CLI Converter! This tool acts like a "language-jelly" translator that automatically converts terminal commands between Windows (CMD / PowerShell) and Linux (Mac / Ubuntu) using the power of AI (OpenRouter / MiniMax) 🚀

---

## ✨ Why use this tool? (Great for beginners / Non-IT people)

- 🧠 No need to memorize hard commands: Type what you know and the AI will convert it for you.
- 🌐 Cross-platform conversion: Convert commands between `PowerShell`, `CMD`, and `Bash (Linux)` in one go.
- 🇹🇭 Explanations in Thai (and short human-readable descriptions): The AI doesn’t just convert commands — it also gives a short explanation of what the command does and whether it’s safe.
- 🛡️ Smart routing: Multiple API backends supported; if the first fails, it automatically falls back to a backup.
