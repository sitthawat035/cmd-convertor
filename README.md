# 🔄 AI CLI Converter - "วุ้นแปลภาษา" สำหรับคนใช้คอม!

เคยเจอปัญหานี้ไหมครับ? ไปเจอสอนในเน็ตให้พิมพ์คำสั่ง `ls` หรือ `rm -rf` (ของฝั่ง Linux) แต่คอมเราเป็น Windows พอพิมพ์ตามแล้วคอมด่ากลับมาว่า "Command not found!" 😫

**หมดปัญหานั้นทันทีด้วย AI CLI Converter!** เครื่องมือชิ้นนี้เปรียบเสมือน "วุ้นแปลภาษา" ที่จะช่วยคุณแปลงคำสั่งหน้าจอดำๆ ระหว่าง **Windows (CMD / PowerShell)** และ **Linux (Mac / Ubuntu)** แบบอัตโนมัติ โดยใช้ขุมพลังความฉลาดของ AI (OpenRouter / MiniMax) 🚀

---

## ✨ ทำไมต้องใช้เครื่องมือนี้? (เหมาะมากสำหรับมือใหม่ / Non-IT)

- 🧠 **ไม่ต้องจำคำสั่งยากๆ:** พิมพ์สิ่งที่คุณรู้ แล้วเดี๋ยว AI แปลงให้เอง!
- 🌐 **ข้ามค่ายได้สบาย:** แปลงคำสั่งไปมาระหว่าง `PowerShell`, `CMD` และ `Bash (Linux)` ให้ครบในรอบเดียว
- 🇹🇭 **อธิบายภาษาคน (ไทย):** ไม่ใช่แค่แปลคำสั่งให้ แต่ AI จะช่วย "อธิบาย" สั้นๆ ด้วยว่าคำสั่งนั้นเอาไว้ทำอะไร ปลอดภัยไหม
- 🛡️ **ระบบฉลาดล้ำ (Smart Routing):** ต่อ API ได้หลากหลาย ถ้าอันแรกพัง มันสลับไปหาอันสำรองให้เอง!

## 🚀 เริ่มต้นใช้งานง่ายๆ ใน 3 ขั้นตอน

ไม่ต้องเป็นโปรแกรมเมอร์ก็ใช้ได้! แค่ทำตามนี้ครับ:

### 1. ติดตั้งของที่ต้องใช้
เปิด Terminal / Command Prompt แล้วพิมพ์คำสั่งนี้เพื่อดาวน์โหลดส่วนเสริม:
```bash
pip install requests
```

### 2. ใส่กุญแจ AI (API Key)
เครื่องมือนี้ใช้ความฉลาดของ AI จึงต้องมีกุญแจส่วนตัว:
1. ก๊อปปี้ไฟล์ `.env.example` แล้วเปลี่ยนชื่อเป็น `.env` (จุดนำหน้าคำว่า env)
2. เปิดไฟล์ `.env` ขึ้นมาด้วย Notepad
3. เอา API Key ที่คุณมี (เช่น ของ OpenRouter หรือ MiniMax) ไปแปะหลังเครื่องหมายเท่ากับ `=` แล้วกดเซฟ

### 3. เปิดใช้งานเลย!
แค่พิมพ์คำสั่งนี้ใน Terminal:
```bash
python cli_converter.py
```
*(ถ้าตั้ง Alias ไว้แบบเครื่องผม ก็แค่พิมพ์ `cmdconv` จบเลย! 😎)*

## 🎮 ตัวอย่างความว้าว

ลองพิมพ์คำสั่งพวกนี้ดูสิ แล้วดูเวทมนตร์ของ AI:

| คุณลองพิมพ์ | สิ่งที่ AI จะแปลงให้ |
|-------------|---------|
| `ls -la` | AI จะบอกว่านี่ของ Linux และแปลงเป็น `Get-ChildItem -Force` สำหรับ Windows ให้ |
| `dir /s` | AI จะรู้ว่านี่คือของ Windows และแปลงเป็น `ls -R` หรือ `find` ของ Linux ให้ |
| คำสั่งสุดโหด | หา PID โคตรลึก AI ก็หั่นโครงสร้างเป็น PowerShell Function ให้ได้! |

## 💡 โครงสร้างสำหรับสาย Tech (For Nerds)

ถ้าคุณพอรู้เรื่องโค้ด โปรเจกต์นี้เขียนแบบ **Modular Architecture** แยกไฟล์ชัดเจน เอาไปต่อยอดง่ายมาก:
- `cli_converter.py`: ประตูทางเข้าโปรแกรม (Entry Point) สั้นๆ คลีนๆ
- `core.py`: หัวใจหลักในการคุยกับ AI และสร้าง Prompt
- `config.py`: ระบบสลับสับเปลี่ยน API อัจฉริยะ (Local > Tailscale > Direct)
- `prompt.py`: กล่องเก็บจิตวิญญาณ AI (System Prompt)
- `utils.py`: ตัวตกแต่งหน้าตาโปรแกรมให้สวยงาม
